# Обучение "Docker"

## 🐳 Описание
В этом проекте я учусь использовать **Docker**, изучаю то как он работает, как развернуть на нем БД - PostgreSQL, ClickHouse. README файл опишет некоторые моменты связанные с Docker.

## 👀 Зачем изучать Docker
1. **Изоляция приложений** -  каждый контейнер работает в своём окружении, отделённом от других приложений и процессов на хост-системе. Это позволяет избежать конфликтов зависимостей и библиотек.
2. **Портативность** - контейнеры можно запускать на любой машине, которая поддерживает Docker: на локальном компьютере разработчика, сервере в облаке или кластере серверов.
3. **Эффективное использование ресурсов** - в отличие от виртуальных машин, контейнеры не требуют запуска отдельной операционной системы для каждого приложения.
4. **Быстрое развёртывание** - контейнеры можно запускать и останавливать за считанные секунды, что ускоряет процесс разработки, тестирования и развёртывания приложений.
5. **Масштабируемость** - Docker позволяет запускать несколько экземпляров контейнеров на разных серверах, что особенно полезно для облачных и распределённых приложений.
